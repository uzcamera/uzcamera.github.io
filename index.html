使用教程

UZ影视使用教程

UZ影视本身是个空壳软件，需要添加数据源之后才可以正常使用。以下是详细的使用教程：

1. 输入链接添加（推荐）

入口：

uz影视 -> 设置 -> 数据管理 -> 订阅 -> 添加 -> 输入以下链接(二选一或分次全都要) -> 确定(添加过程较慢)

阿里|夸克|UC| VOD 源:

https://github.moeyy.xyz/https://raw.githubusercontent.com/YYDS678/uzVideo-extensions/refs/heads/main/uzAio.json

阿里|夸克|UC|天翼|网盘源:

https://ghproxy.cn/https://raw.githubusercontent.com/proversion2024/uz-extensions/refs/heads/master/uzAio.json
2. 网页添加

在浏览器打开网页（部分网络可能无法打开）：

https://uz-video-five.vercel.app/install.html

找到需要添加的资源，点击添加自动跳转到uz影视并完成添加，如果点了没反应就点特殊添加

注意：需要允许浏览器跳转到外部APP，最好把uz影视打开挂在后台，如果还是无法添加就复制链接，粘贴到以下入口手动添加

入口：

uz影视 -> 设置 -> 数据管理 -> 订阅 -> +

3. 直播源添加

进入 YanG-1989 大佬的直播源分享网站：

https://yang-1989.eu.org/

点击 想要添加的直播类型 自动复制

直播源添加：

uz影视 -> 设置 -> 数据管理 -> 直播源 -> + -> 输入任意名称 -> 输入刚刚复制的链接 -> 确定

uz影视 -> 点击底部小电视 -> 点击顶部小电视 -> 选择添加的直播源 -> 点击右上角切换展示分类

4. 其它设置

推荐扩展：

uz影视 -> 设置 -> 数据管理 -> 推荐扩展 -> 点击扩展 -> 启用推荐主页 -> 重启 app

网盘工具扩展：

uz影视 -> 设置 -> 数据管理 -> 网盘工具扩展 -> 点击扩展 -> 重启 app

5. Cookie 设置

UC | 夸克 | 天翼：

uz影视 -> 设置 -> 数据管理 -> 环境变量 -> 点击右上角按钮 -> 点击 cookie 按钮 -> 选择 uc 或 夸克 -> 登录 -> 点击右上角 -> 点击登录的类型 -> 确定绑定

阿里 (两个都要填！需要手动复制)：

uz影视 -> 设置 -> 数据管理 -> 环境变量 -> 点击右上角按钮 -> 点击 cookie 按钮 -> 选择 阿里 32 -> 扫码登录 -> 复制一串 数字与字母的混合文本 -> 返回 -> 点击 阿里 32 的那个笔 -> 粘贴 -> 确定

uz影视 -> 设置 -> 数据管理 -> 环境变量 -> 点击右上角按钮 -> 点击 cookie 按钮 -> 选择 阿里 280 -> 扫码登录 -> 复制一串 数字与字母的混合文本 -> 返回 -> 点击 阿里 280 的那个笔 -> 粘贴 -> 确定
